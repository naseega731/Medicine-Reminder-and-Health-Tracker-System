 Login and Dashboard UI Design

## Objective

To design the Login and Dashboard user interfaces for the Medicine Reminder and Health Tracker System using Python Tkinter.

## Login Screen Design

The Login Screen is the entry point of the application. Users can enter their username and password to access the system.

### Features

* Username input field
* Password input field
* Login button
* User-friendly interface

## Dashboard Screen Design

The Dashboard is displayed after successful login. It provides access to all major modules of the system.

### Features

* Medicine Management
* Reminder Management
* Health Tracker
* Reports and Analytics
* Logout option

## Tools Used

* Python
* Tkinter

## Expected Outcome

The Login and Dashboard screens provide a simple and easy-to-use interface for users to navigate the Medicine Reminder and Health Tracker System.
